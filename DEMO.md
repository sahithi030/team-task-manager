# Team Task Manager - Demo Guide

This document provides a comprehensive guide for demonstrating the Team Task Manager application.

The Team Task Manager is a full-stack application that enables teams to collaborate on projects and manage tasks efficiently. This demo covers all major features and user workflows.

### 1. Application Introduction (30 seconds)
- Show the login page with modern UI
- Highlight the application name and branding
- Mention the tech stack (React, Node.js, MongoDB)

### 3. Dashboard Overview (45 seconds)
- Show the main dashboard with statistics
- Highlight key metrics (total tasks, my tasks, projects, overdue)
- Show recent tasks and projects widgets
- Demonstrate quick action buttons

### 4. Project Management (60 seconds)
**Create New Project:**
1. Click "New Project" button
2. Fill in project details (name, description, priority)
3. Show project creation success
4. View project in projects list

**Project Features:**
1. Click on a project to view details
2. Show project statistics and task breakdown
3. Demonstrate member management
4. Show project settings and options

### 5. Task Management (60 seconds)
**Create New Task:**
1. Click "New Task" from dashboard or project
2. Fill in task details (title, description, assignee, priority, due date)
3. Show task creation and assignment
4. Demonstrate task status updates

**Task Features:**
1. View task details with comments
2. Update task status (Todo → In Progress → Completed)
3. Show task filtering and search
4. Demonstrate task assignment to team members

### 6. Team Collaboration (45 seconds)
**User Management:**
1. Navigate to Users section (admin only)
2. Show user list with roles
3. Demonstrate adding new team members
4. Show role-based permissions

**Project Collaboration:**
1. Add members to a project
2. Assign different roles (admin/member)
3. Show how team members can view and work on tasks
4. Demonstrate commenting on tasks
